# 34DEV Scripting - Tanıtım Sitesi (Static)

Bu repo sadece tanıtım amaçlı tek sayfa web sitesidir. Login/kayıt yoktur.

## GitHub Pages ile yayınlama
1) Bu repo'yu GitHub'a yükle
2) Repo > Settings > Pages
3) "Build and deployment" altında:
   - Source: Deploy from a branch
   - Branch: main / (root)
4) Save
5) GitHub sana bir URL verecek (örn: https://kullaniciadi.github.io/repo-adi)
